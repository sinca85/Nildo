# Soy Nildo Landing

Landing estática pensada para GitHub Pages.

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Publicación rápida en GitHub Pages
1. Crear un repositorio nuevo.
2. Subir todos estos archivos a la raíz del repo.
3. Ir a **Settings > Pages**.
4. En **Build and deployment**, elegir **Deploy from a branch**.
5. Seleccionar la rama `main` y la carpeta `/root`.
6. Guardar.

## Para sumar recetas
En `script.js`, dentro del objeto `RECIPES`, agregar nuevas entradas siguiendo la misma estructura.
Luego duplicar una card en `index.html` dentro de `.recipes-grid`.
